# 后端

后端工作区是 `infoq-scaffold-backend`。这一栏只讲当前仓库真实存在的结构、配置、插件和运行路径，不复述通用 Spring Boot 教程。

当前系统监控接口已经覆盖 `ServerController` 和 `DataSourceController`；连接池监控基于本项目 Hikari 原生实现，不依赖 Druid 控制台。
