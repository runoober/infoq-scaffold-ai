# 管理端

本项目同时维护 Vue 3 + Element Plus 和 React 19 + Ant Design 两套管理端。正文重点是“共享了什么”和“各自如何实现”，而不是把两边写成两份互不相关的产品文档。

系统监控能力现已包含在线用户、日志、任务调度、缓存、服务监控和 Hikari 原生连接池监控。
