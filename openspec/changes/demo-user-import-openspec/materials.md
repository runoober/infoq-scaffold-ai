# Materials: demo-user-import-openspec

## 素材范围

本示例需要准备的素材类型：

- 导入模板列头示例
- 上传说明文案
- 成功/失败提示
- 图标建议

## Mock Data

- 用户名：`zhangsan`
- 昵称：`张三`
- 手机号：`13800000000`
- 邮箱：`zhangsan@example.com`
- 部门：`研发中心`
- 角色：`普通用户`

建议示例数据：

- 1 行合法用户
- 1 行重复用户名示例
- 1 行缺少手机号示例
- 默认 `updateSupport=false`

## 占位文案

- 标题：`用户导入`
- 副标题：`请先下载模板，按格式填写后再上传`
- 空状态：`暂未选择导入文件`
- 错误态：`导入失败，请检查模板内容后重试`
- 成功提示：`导入完成，请查看结果明细`

## 图标与视觉建议

- 主图标：上传图标，沿用当前组件库默认上传语义图标
- 状态图标：成功使用勾选，失败使用警示
- React 端：沿用 Ant Design 上传拖拽区和轻量提示样式
- Vue 端：沿用 Element Plus 上传图标与提示区域样式

## 交互反馈文案

- Loading：`文件上传中，请稍候`
- Confirm：`确认导入选中的用户数据吗？`
- Warning：`开启覆盖后，已存在用户数据将按模板更新`
- Error：`仅允许导入 xls、xlsx 格式文件`

## 替换说明

- 上线前必须替换：示例手机号、邮箱、部门、角色等 mock 数据
- 可以继续作为开发占位：示例用户名与空态文案
